# Boolean-expression-parser
Expresion:
E - equivalence
I - implication
C - conjuction
D - disjunction
n - negation



You should enter the expersion without spaces, like that:
Original expresion:
[(p ∧ q) ⇒ r] ⇔ [p ⇒ (q ⇒ r)]
Input expresion:
E(I(C(p,q),r),I(p,I(q,r)))
You should use 'p' when you have one variable, 'p', 'q' when two and 'p', 'q', 'r' when 3.
